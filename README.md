# Mocha

## Install Mocha
 - npm install -g mocha
 
## Install Should
 - npm install --save-dev should
 
## Install Assert
 - npm install --save-dev assert
 
## Install Mochawesome
 - npm install --save-dev mochawesome

## Run unit test
 - mocha "filename.js"

## Create test report
 - mocha "filename.js" --reporter mochawesome
